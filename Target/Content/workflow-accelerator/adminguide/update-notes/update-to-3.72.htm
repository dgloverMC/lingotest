Updating to Signavio Workflow Accelerator 3.72

If you use Workflow Accelerator in combination with Signavio Process Manager, you must update Process Manager to version 12.3.

The Apache Tomcat Team announced that the support for Apache Tomcat 8.0.x has ended June 30, 2018. That's why Workflow Accelerator now only supports Apache Tomcat 8.5.x. If you are upgrading from Tomcat 8.0.x or older to Tomcat 8.5.x and configured SSL / TLS previously, then you will have to adjust your settings. The official Tomcat 8.5.x documentation explains how to configure SSL / TLS:

https://tomcat.apache.org/tomcat-8.5-doc/ssl-howto.html

The Process Manager admin guide also contains instructions on how to setup SSL with Tomcat:

https://docs.signavio.com/adminguide/editor/en/installation/https.html
