Backup the configuration files

During the first setup you have modified some configuration files and added the Workflow Accelerator license file. In order to preserve those changes, you have to copy those files to another location before updating the Workflow Accelerator system. You can find those files in the directory $TOMCAT_HOME/webapps/ROOT/WEB-INF/classes.

effektif-onpremise.properties Workflow Accelerator configuration file
logback.xml configuration file for Workflow Accelerator logs
license.xml Workflow Accelerator license file

Simply copy the files mentioned above to another location, e.g. $TOMCAT_HOME/webapps.

If you installed the Workflow Accelerator Script Engine, you modified the configuration file configuration.onpremise.js of the script engine as well. You can find the configuration file directly within the $SCRIPT_ENGINE_HOME directory. Please copy it as well to another location before you go on.
