Configuring approval workflows with Signavio Workflow Accelerator (optional)

Signavio provides Signavio Workflow Accelerator to customers who purchased a corresponding subscription. If your organization wants to use the approval workflows feature, you need to install Signavio Workflow Accelerator on your server.

Refer to the Signavio Workflow Accelerator administrators guide for more information about the system requirements and the installation process.

Before installing Signavio Workflow Accelerator, please ensure your Signavio version is compatible with the Signavio Workflow Accelerator version you plan to install. You find a corresponding compatibility list here.
