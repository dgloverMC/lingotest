Registering users

To register a user, open the following URL in your browser:

http://<IP address or DNS of the Signavio system>/p/register

Fill out the form and click Register. Repeat this step for each user you want to register. Users login by entering their email address and a personally chosen password.

A user’s email address is a unique ID. It is not possible to change a user’s email address after its registration!

The first registered user is automatically added to the special user group Administrators. You can add more users manually to the group. Users who are member of the Administrators group operate with extended rights that allow managing user groups or adding modeling guidelines. Therefore, the first user is able to add other users to the administrators group.
