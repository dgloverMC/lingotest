Configure image server (optional)

The configuration procedure of the image server is similar to the configuration of the Solr server.

Follow the instructions in Configuring the Apache Solr server (optional) .

This time, move the folder imageservice from the webapps directory of the Signavio Tomcat to the webapps directory of the image service Tomcat and add the following line to the configuration.xml file: 

<imageServiceHost>http://localhost:8080</imageServiceHost>

Replace the URL http://localhost:8080 with the URL of your image server.
