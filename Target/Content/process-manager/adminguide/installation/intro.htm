Installation

The following chapter provides step-by-step instructions on how to install Signavio Process Manager. You need to have the ZIP archive available that contains Signavio Process Manager's installation files, as well as a license file.

In the installation instructions, the term $TOMCAT_DIR is used in command line statements to denote the path to the Apache Tomcat instance which should point to the directory above the bin and webapps directories and which runs the Signavio Process Manager web application. To be able run the commands right away after copying & pasting them, you can set a corresponding environment variable.
