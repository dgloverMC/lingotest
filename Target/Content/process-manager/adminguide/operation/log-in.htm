Accessing Signavio Process Manager as a user

You access Signavio Process Manager with any modern web browser (see https://www.signavio.com/browser-compatibility/ for a list of supported browsers).

In case you have problems connecting to Signavio Process Manager, make sure your browser's network and proxy settings are correct.

To login to the Signavio web application you need to have a registered user (and their email address and password).

Collaboration Hub without single-sign-on allows any user who can access the Signavio server's URL to view published diagrams in your browser. If some users within your network shouldn't have access to Collaboration Hub, you have to manage these access restrictions on network level.

In contrast, Collaboration Hub with single-sign-on allows you to define user and group specific access rights for each diagram based on your Active Directory service.
