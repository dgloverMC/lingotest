Moving the server to new hardware

Moving the server to a different hardware is no problem at all. Copy your license file and create a dump of your database. Now, install the system from scratch on your new machine and import the database dump there. Ensure all external systems are accessible from the new server (email server, database, Active Directory, SharePoint etc.).

Always backup your data before you migrate. On shut down, stop the Tomcat server before stopping the database system.
